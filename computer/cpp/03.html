# 三. 异常处理机制

检查异常：使用try语句块

抛出异常：使用throw语句块

捕捉异常：使用catch语句块

格式：

  • throw 表达式；

  • try {被检查的语句块}

  • catch (类型1 参数1)

    {进行异常处理的语句}

    catch (类型2 参数2)

    {进行异常处理的语句}

    catch (……)

    {进行异常处理的语句}
